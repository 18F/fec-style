7 #type-styles Type Styles

For classes that define type styles Prefix these styles with t-

7.1 #type-styles.display Display

Large type with top and bottom borders for major page headings. In order to have rules, apply .t-ruled

Examples 
.t-ruled
Apply top and bottom rules to the heading

Some great heading

<h2 class="t-display [modifier class]">Some great heading</h2>

7.2 #type-styles.display-sans Display sans

Used for showing large numbers. Sans-serif and no border

Example
$2,010 
<span class="t-display--sans">$2,010</span>

7.5 #type-styles.ruled-headings Ruled Headings

Adds top and bottom borders; color changed by u-font-color

Examples 
.t-ruled
1px top and bottom rule

Another heading

.t-ruled--bold
2px weight borders

Another heading

.t-ruled--bottom
Only rule on the bottom

Another heading

.t-no-rules
Removes a rule on an item that has one by default

Another heading
