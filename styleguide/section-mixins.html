5 #mixins Utility Mixins

5.1 #mixins.font-colo form-color()

  • $background-color
    The color of the background of the inputs
  • $contrast-color
    Border and font colors
  • $focus-color
    Focus color
  • $icon-color
    A string that will be appended to the background-image url for checkboxes (e.g. 'primary', 'secondary')

Used to set the color of the form elements

5.2 #mixins.font-color font-color()

  • $color
    The color of the font and (if applicable) border-color

Used to set the color of all child elements

5.3 #mixins.visually-hidden u-visually-hidden()

Used to visually hide components but keep them visible to screenreaders
