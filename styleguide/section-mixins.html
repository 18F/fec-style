2 #mixins Utility Mixins

2.1 #mixins.background-image u-bg-background-image()

  • $image
    Name of the image file. It will create two calls, one to a png and one to an svg
  • $position:
    Will set the background-position property, defaults to 50% 50%

Generates everything you need for a background image

2.2 #mixins.bg-primary u-bg--primary()

Used to set generate all of the styles for a bg--primary element

2.3 #mixins.bg-secondary u-bg--primary()

Used to set generate all of the styles for a bg--primary element

2.4 #mixins.font-colo u-form-color()

  • $background-color
    The color of the background of the inputs
  • $contrast-color
    Border and font colors
  • $focus-color
    Focus color
  • $icon-color
    A string that will be appended to the background-image url for checkboxes (e.g. 'primary', 'secondary')

Used to set the color of the form elements

2.5 #mixins.font-color u-font-color()

  • $color
    The color of the font and (if applicable) border-color

Used to set the color of all child elements

2.6 #mixins.truncate u-truncate()

Truncate text to a single line with an ellipsis Thanks to http://css-tricks.com/snippets/css/truncate-string-with-ellipsis/

2.7 #mixins.visually-hidden u-visually-hidden()

Used to visually hide components but keep them visible to screenreaders. Also includes a utility class .u-visually-hidden.
