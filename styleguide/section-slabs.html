4 #slabs Slabs

Slabs are full-width blocks of color. Slabs should contain containers to center content. Slabs can can have background color styles applied

Example
This is a neutral slab
This is a primary slab
This is a secondary slab 
<div class="slab bg--neutral">
  <div class="container">
    This is a neutral slab
  </div>
</div>
<div class="slab bg--primary">
  <div class="container">
    This is a primary slab
  </div>
</div>
<div class="slab bg--secondary">
  <div class="container">
    This is a secondary slab
  </div>
</div>
