7 #layout Layout

These are basic grid and color styles to be applied to components

7.1 #layout.backgrounds Backgrounds

  • .bg--neutral
    Ivory background with black text
  • .bg--primary
    Federal Blue background with aqua text
  • .bg--secondary
    Crimson background with orange text

Background styles can be applied to any component and will apply background-color and child element colors Uses the u-bg--*() mixins to set colors

7.2 #layout.grid Grid

  • .container
    Used to set group grid items

7.3 #layout.slabs Slabs

Slabs are full-width blocks of color. Slabs should contain containers to center content. Slabs can can have background color styles applied
