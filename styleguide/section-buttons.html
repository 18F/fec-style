1 #buttons Buttons

Examples
Default styling 
.button
The basic styled button that contains the properties that get extended by more specific classes 
<button>Basic button</button>

1.1 #buttons.icon-buttons Icon Buttons

Icon color matches text color Small vertical line between icon and text. Buttons that have “forward” actions have icons to the right of the text, and those with “back” actions have icons to the left.

Example 
<button class="button--icon button--secondary"><img alt="arrow-left" src="../img/i-arrow-left--secondary.svg">Previous</button>
<button class="button--icon button--secondary button--icon--right">Next<img alt="arrow-right" src="../img/i-arrow-right--secondary.svg"></button>
<button class="button--icon button--secondary"><img alt="charts" src="../img/i-chart--secondary.svg">View charts</button>
