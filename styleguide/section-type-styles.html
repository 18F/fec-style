14 #type-styles Type Styles

For classes that define type styles Prefix these styles with t-

14.1 #type-styles.display Display

Large type with top and bottom borders for major page headings. In order to have rules, apply .t-ruled

Examples
Default styling

Some great heading

.t-ruled
Apply top and bottom rules to the heading

Some great heading

<h2 class="t-display [modifier class]">Some great heading</h2>

14.2 #type-styles.display-sans Display sans

Used for showing large numbers. Sans-serif and no border

Example
$2,010 
<span class="t-display--sans">$2,010</span>
