8 #functions Utility Functions

Various useful functions.

8.1 #functions.convert-units Unit conversion

When using the FEC styleguide with other frameworks, it might be useful to scale and convert the FEC component sizes. Use these variables to adjust how the styleguide is built.

  • $px-only
    (bool) convert rem units to px
  • $base-size-px
    (integer) sets the base size for a rem unit (only take effect when $px-only is true)

8.1.1 #functions.convert-units.function u($values)

Convert and/or scale rem units. This should be used on all properties related to component size. This allows FEC styleguide to scale component sizes properly.

Inspired by https://github.com/saxinte/rem-to-px-revisited

  • $values
    single unit or list of units e.g. 1rem 1.2rem

8.2 #functions.parse-int u-parse-int($n)

Parses the integer part from a value with unit.

  • $n
    the value to parse e.g. 2em

8.3 #functions.scale-to-px u-scale-to-px($n)

Scales the value according to $base-size-px and appends the px unit.

  • $n
    value to scale to px
