9 #mixins Utility Mixins

9.1 #mixins.background-image u-bg-background-image()

Generates everything you need for a background image

  • $image
    Name of the image file. It will create two calls, one to a png and one to an svg
  • $position:
    Will set the background-position property, defaults to 50% 50%

9.2 #mixins.bg-dark u-bg--primary()

Used to set generate all of the styles for a bg--dark element

9.3 #mixins.bg-primary u-bg--primary()

Used to set generate all of the styles for a bg--primary element

9.4 #mixins.bg-secondary u-bg--primary()

Used to set generate all of the styles for a bg--secondary element

9.5 #mixins.font-color u-font-color()

Used to set the color of all child elements

  • $color
    The color of the font and (if applicable) border-color

9.6 #mixins.icon() u-icon()

Used for creating a button or single icon with a background image with no text

  • $icon:
    Name of the icon to use
  • $color:
    Color variable the icon should be

9.7 #mixins.icon-bg u-icon-bg()

Most basic mixin for using a variable svg as a background image

9.8 #mixins.icon-button u-icon-button()

  • $icon
    Name of the icon variable. Will be inlined as a data uri
  • $color
    Color variable the icon should be
  • $side
    Icon on the left or right? Defaults to left
  • $width
    Width of the icon, defaults to 1.5rem

9.9 #mixins.icon-circle u-icon-circle()

Generates a circle with a background color and an icon within that circle Can be used in pseudo-elements to create a quasi-background image

9.10 #mixins.truncate u-truncate()

Truncate text to a single line with an ellipsis Thanks to http://css-tricks.com/snippets/css/truncate-string-with-ellipsis/

9.11 #mixins.visually-hidden u-visually-hidden()

Used to visually hide components but keep them visible to screenreaders. Also includes a utility class .u-visually-hidden.
